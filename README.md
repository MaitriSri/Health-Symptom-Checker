# 🩺 Health Symptom Checker

## 📌 Project Description

This project is a machine learning-based web application that predicts possible diseases based on user-selected symptoms.

## 🎯 Objective

To help users get a basic idea of possible illnesses based on symptoms and provide simple advice.

## ⚙️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## 🧠 Features

* Symptom-based disease prediction
* Simple and interactive UI
* Provides basic health advice

## 📚 Learning Outcomes

* Learned machine learning basics
* Built a real-world application
* Understood classification algorithms

## ⚠️ Disclaimer

This project is for educational purposes only and should not be used for medical diagnosis.
